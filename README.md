# 多廠協同管理決策室

整合跨廠區產能、庫存與訂單資訊，實現集團多廠協同生產調度之平台。

- 原始系統：多廠協同管理平台（Multi-Plant Collaboration Platform）
- 產業分類：智慧製造｜製造管理
- 使用部門：集團營運、生管、供應鏈管理部門
- 工作流程：各廠數據彙整 → 集團產能/庫存分析 → 跨廠分配決策 → 執行下達各廠 → 績效比較回饋
- AI 應用：AI建議最適跨廠訂單分配方案、智慧預測集團級產能瓶頸
- 主要 KPI：跨廠產能利用均衡度、庫存調撥效率、集團交期達成率
- 代表產品：SAP IBP、Kinaxis RapidResponse、o9 Solutions
